# flutter_application_practice

First repo for Flutter Class | © SkillBridge Institute of Technology

This project is a clone of two Instagram pages: the **Home/Feed page** and the **Profile page**.
* **Home page**: Contains stories scrollable horizontally and posts vertically.
* **Profile page**: Contains profile info such as picture, name, and other details.

The project started on day one and is an ongoing multifunctional Instagram clone.
